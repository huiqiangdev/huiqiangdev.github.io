设计模式-组合模式

什么是组合模式？
以表示部分和整体的层次结构，是的用户对单个对象和组合对象的使用具有一致性.
例如UIView中的addSubView方法.

使用场景：

  • 场景一：表示对象整体和部分结构的时候。
  • 场景二：从一个整体中能够独立出部分模块或者功能的

组合模式中划分的角色：

  • 角色一：抽象根节点（Component:组合中的对象申明接口）对组合对象进行抽象
  • 角色二：具体节点（Composite）
  • 角色三：叶子（末端）节点（Leaf:没有子节点，作用：在组合中定义节点对象行为）类似于Swift中final Class 不可以继承的类。
